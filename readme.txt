How to Play:

Your objective is to collect all 5 portal pieces that is placed 
within each level. Each level will have its own enemy type with
its own unique finite state machines. You can only get pass through them
through utilizing mechanics. There will be a total of 5 levels
and in order to win, u need to get to the exit to level 5 and
bring all portal pieces together to escape.


Player Controls

-Arrow Key Right (Move right)
-Arrow Key left (Move left)
-Arrow key Down(Move Down)
- Spacebar ( To jump)
- Z to use Timestop (if u have time orb)
- X to teleport(if u have teleport control)
- C to clone (If you have clone orb)
- U to use ultimate (if ultimate is at 100%)
